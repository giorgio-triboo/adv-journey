Soglie Margine e Scarto

Configura le soglie per la colorazione nella vista Marketing. I valori di margine % e % scarto vengono evidenziati in rosso, arancione o verde in base alle soglie impostate.

Margine %

Il margine è migliore quando è alto. Rosso = fino a soglia rossa, Arancione = tra le due soglie, Verde = sopra soglia verde.

Valori ≤ questa soglia = rosso

Valori ≥ questa soglia = verde

% Scarto (ingresso e uscita)

Lo scarto è migliore quando è basso. Verde = fino a soglia verde, Arancione = tra le due soglie, Rosso = sopra soglia rossa.

Valori ≤ questa soglia = verde

Valori ≥ questa soglia = rosso

Torna a Marketing

Legenda colori:
Rosso = critico  |  Arancione = attenzione  |  Verde = ok
