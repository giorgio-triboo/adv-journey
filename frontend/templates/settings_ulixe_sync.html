Scraping automatico RCRM da Google Sheet

Questa sezione gestirà lo scraping automatico del file RCRM Ulixe direttamente da Google Sheet, con job schedulabile via cron e avviabile manualmente da qui.

Sorgente dati: Google Sheet RCRM Ulixe
Nome file generato: {mese}-ulixe-rcrm
Range colonne usato: A1:G24 (uguale al template rcrm-0326.csv)

Il job legge tutti i file rcrm-*.csv presenti in backend/exports/ulixe_temp, deduce il mese dal nome e per il mese selezionato sovrascrive completamente i dati in ulixe_rcrm_temp.

Upload RCRM mensile (Ulixe)

Carica il file RCRM Ulixe per un mese specifico. Il caricamento sovrascrive completamente i dati esistenti per il mese selezionato per evitare duplicazioni.

Esempio: 2026-03 per il file rcrm-0326.csv.

Usa il template come rcrm-0326.csv con colonne IDMessaggio e RCRM.
